# AgentGuild

A collection of AI agent skills for blockchain and web3 development, featuring ERC-402 payment-gated API tools.

## Features

- **Agent Guild** - AI agent on Base Mainnet with 31 API tools
- **ERC-402 Payment System** - USDC-based access control with fixed receiver
- **Data-Driven Pricing** - Costs based on actual Base Mainnet gas analysis
- **Four-Tier System** - Free, Lightweight ($0.01), Heavyweight ($0.10), Ultra-Heavy ($0.50)

## Skills

| Skill | Description |
|-------|-------------|
| [agent-guild](skills/agent-guild/SKILL.md) | Create, deploy, and manage AI agents on Base Mainnet with ERC-402 payments |

## Payment System

All paid API tools require ERC-402 payment in USDC on Base Mainnet.

**Payment Receiver:** 0x21c2b22b7dD5634Af6fc0bc5489965bb906a9959

### Cost Structure

| Category | Service Fee | Total Cost Range |
|----------|-------------|------------------|
| Free | $0.00 | $0.00 |
| Lightweight | $0.01 USDC | $0.018-0.05 USDC |
| Heavyweight | $0.10 USDC | $0.14-0.18 USDC |
| Ultra-Heavy | $0.50 USDC | $0.58-2.50+ USDC |

All costs include Base Mainnet gas fees

## Usage

1. Clone the repository
2. Load the desired skill using your AI assistant
3. Follow the skill instructions to deploy and configure
4. Make ERC-402 payments in USDC for paid tools
